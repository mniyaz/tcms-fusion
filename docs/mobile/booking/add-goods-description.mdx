# 📦 Quick Guide: Adding Goods Details

**Step 1** : Open the Goods screen by completing booking details and tapping Continue, or tap + Add Goods.

**Step 2** : Enter item details such as Goods Name, Ref No, Weight, Volume, Size, and Pieces (Pcs).

**Step 3** : Select the Package Type and choose From Pickup and To Delivery locations.

**Step 4** : (Optional) Tap + Add Goods to add more items or ✕ to remove any item.

**Step 5** : Tap Continue to proceed or Save to keep the details and go back.
