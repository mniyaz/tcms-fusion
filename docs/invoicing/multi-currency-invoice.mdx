# Multi-Currency Invoice

This guide explains how to generate invoices using a customer’s selected currency instead of the company’s base currency.

--- 1. Navigate to **HR → Customer**, open the customer, select the desired **Currency**,
and click **Save**.

2. Under **Invoicing**, generate the invoice for this customer following the standard process.

3. In the invoice form and print view, amounts are shown in the customer’s selected currency with the symbol or code, instead of the base currency.
