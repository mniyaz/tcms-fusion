Go to **Booking Management -> Create Booking**

### Create Dedicated Booking 

-Fill in the customer details. 

-In the Order Type drop-down, choose **DEDICATED.** 

-Select **Driver and Truck**  

-Enter the pickup and delivery details, including truck tonnage,price and type. 

-Click the **SUBMIT** button to save the booking. 
 

Now this **booking will appear in driver app**, the price(if entered) will automatically apply for every booking made.
